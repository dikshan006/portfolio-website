# Netflix-Style Portfolio Website

## Overview
This is a Netflix-inspired personal portfolio designed to showcase my AI, automation, and analytics projects with a fast, responsive, and recruiter-friendly UI.

It acts as a central hub for live demos, projects, and contact information.

## What It Does
- Displays projects in a Netflix-style grid with previews and sections  
- Links directly to live applications and GitHub repositories  
- Provides a clean, mobile-first overview of my technical work  

## Tech Stack
- Frontend: Next.js, React  
- Styling: Tailwind CSS  
- Hosting: Vercel (custom domain + SSL)  
- Tooling: GitHub, CI/CD  

## Key Features
- Responsive layout optimized for desktop and mobile  
- Project cards with descriptions, tech stack, and live links  
- Lighthouse mobile performance score 92+  
- Sub-150ms initial load time on Vercel
