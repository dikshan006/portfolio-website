# Netflix-Style Portfolio Website

This is my personal portfolio, styled like a Netflix interface, showcasing live demos of my AI, automation, and analytics projects.

## What It Does

- Displays my projects in a Netflix-style grid with sections and hover previews.
- Links directly to live apps (KenZen AI, Finance Dashboard, etc.) and GitHub repos.
- Optimized for mobile performance, SEO, and fast load times.

## Tech Stack

- Frontend: Next.js, React
- Styling: CSS / styled components / Tailwind (adjust to what you actually use)
- Hosting: Vercel (custom domain + SSL)
- Tooling: GitHub CI/CD

## Key Features

- Responsive layout with smooth navigation and project carousels.
- Lighthouse mobile performance score 92+ and sub-150ms initial load (on Vercel).
- Sections for experience, projects, and contact.
